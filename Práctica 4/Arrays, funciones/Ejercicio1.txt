<?php
$a = array( 'color' => 'rojo',
 'sabor' => 'dulce',
 'forma' => 'redonda',
 'nombre' => 'manzana',
 4
 );
?>

<?php
$a['color'] = 'rojo';
$a['sabor'] = 'dulce';
$a['forma'] = 'redonda';
$a['nombre'] = 'manzana';
$a[] = 4;
?> 

Ambos crean el mismo array.
El primero usa la sintaxis compacta de inicialización.
El segundo lo construye paso a paso mediante asignaciones.
Ambos son válidos y equivalentes.
